# 我的故乡 shangri-la

见证家族起起落落的老房子♥️♥️♥️

纳帕海俯视角 ---- 与对面的石卡雪山隔 “海” 相望

飞来寺 ---- 去朝拜神山卡瓦格博路上必经的寺庙，朝拜前需要至此烧香祈福，意为 “取钥匙”

舞动的经幡 ---- 正月初十，全家到香格里拉市区东南 10 公里处的大宝寺烧香祈福

纳帕海俯视角 ---- 雪景下的纳帕海早已变成白雪茫茫的童话世界

# 版纳之行

告庄西双景大金塔夜景 ---- 算是著名打卡点了，底下就是赶摆夜市

告庄西双景大金塔夜景仰视 ---- 曾一度成为我手机锁屏壁纸

市允景洪澜沧江大桥 ---- 是一座澜沧江上的老桥，目前还能通车游行

西双版纳大桥 ---- 与老桥形成对比，在桥上可以清晰的看到江两岸景色

龙舟广场前的一群假大象

告庄西双景 ---- 打击毒贩电影里经常看到的建筑风格🤦‍♂️🤷‍♂️，不过有一说一，建筑真好看

# 昆明 “荒野求生” 记

云南海埂会堂 ---- 这滤镜以及蓝的出彩的天空搭配庄严的建筑勾勒出的画面绝了

海埂大坝 ---- 建筑人看到如此构图都只能在心底默默念一句 “卧槽，牛逼”

昆明植物园 - 扶荔宫 ---- COP15《生物多样性公约》缔约方大会第十五次会议室外主题展区。

# 普洱工作 + 旅记

思茅港镇 ---- 眼前就是澜沧江，对面就是澜沧县，每天都有络绎不绝的人来回摆渡

云上舍下 ---- 位于哀牢山山腰的一家小众 “网红” 民宿，只不过我好奇为啥镇沅这种八竿子和藏族不沾边的地会有刻有藏语的石头成列在内（哈哈哈可能老板是藏文化迷弟）

云缦倚象山旅游汽车营地 ---- 帐篷式的露天营地，周围都是茶山以及采茶的茶农，土豪打卡必备啊，可惜俺人穷住不起

西盟佤族自治县 ---- 这个 “袖珍县” 主要以佤族为主，西边与缅甸接壤，所以县城内建筑多以佤族风格为主

西盟佤族自治县 ---- 这个 “袖珍县” 主要以佤族为主，西边与缅甸接壤，所以县城内建筑多以佤族风格为主

西盟佤族自治县 ---- 熙康云舍门口的三根小立柱🎶

西盟佤族自治县

西盟佤族自治县

镇沅县哀牢山保护区 ---- 老爷背后的新农村是搬迁过来的，瞧老爷满脸洋溢着社会主义独有的幸福笑容😊

镇沅县哀牢山保护区 ---- 哀牢山看夕阳西下

孟连县 ---- 孟连宣抚司署内
