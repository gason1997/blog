# docker 镜像打包

1. 使用指令 docker images 查看本地的所有镜像

2. 使用 docker save 导出镜像为 tar 包

打包成功后会在桌面生成一个 web.tar 文件

# xshell 将 web.tar 文件传输给 linux

1. 在 xshell 软件内创建会话连接 linux

2. 点击图表打开 xftp，拖动本地文件 web.tar 到 linux 桌面

# linux 部分

1. 输入 ls 查看所有文件，确保 web.tar 已成功接收

2. docker load 读取（解压）web.tar 文件

3. docker run 运行该镜像
