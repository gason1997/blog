# 项目技术选型

云南民族传统体育数字博物馆开发涉及数据较少且更新任务不频繁，为了超前完成开发任务加快开发效率以及节省工作时间，经过交流与需求验证，最终决定使用 node.js 完成后端接口开发。

后端接口技术选型 - node.js + 🚩Express + postgresql + multer 中间件

软件 - vscode + postman 测试工具 + navicat 数据库

🚩选用 Restful 软件架构风格、设计风格

# 项目接口设计

  • 云南民族传统体育数字博物馆接口设计
    • 后台用户
      • 超级管理员
      • 项目增删改查
      • 用户权限、用户增删改查
    • 普通管理员 (项目增查)
    • 项目数据接口
      • 项目基本信息
      • 项目图片
      • 项目视频
      • 项目位置
      • 项目传承人数据信息
    • 地图数据配置
    • 登录接口

# 数据库表设计

  • 后台用户
  • 项目基本信息（部分）
  • 博物馆介绍表数据
