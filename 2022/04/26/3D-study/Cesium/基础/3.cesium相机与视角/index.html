# 代码详解

  • 创建一个 Cesium Viewer 实例，用于显示地图。
  • 获取当前相机对象。
  • 设置相机的纵横比，这是通过将视图宽度除以高度得到的值来实现的。
  • 设置相机的水平视角角度，使用 Cesium.Math.toRadians 函数将角度转换为弧度表示。
  • 设置相机的垂直视角角度，同样使用 Cesium.Math.toRadians 函数将角度转换为弧度表示。
  • 使用 camera.setView 方法调整相机的位置和朝向。destination 参数设置相机的位置，这里使用了经纬度坐标（-75.59777, 40.03883, 1000）。orientation 参数设置相机的朝向，包括偏航角、俯仰角和翻滚角，这些角度同样使用 Cesium.Math.toRadians 函数转换为弧度表示。

# cesium 的三种视角定位方式

# 过渡定位 - flyTo

这种方式切换视角有个过渡的过程，比较常用

# 直接定位 - setView

这种方式是直接切换到目标视角，比较生硬，没有过渡效果。

# 直接定位 - lookAt

这种方式跟 setView 一样，都是直接切换到目标视角。
