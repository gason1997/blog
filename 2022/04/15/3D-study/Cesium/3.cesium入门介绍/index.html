# Cesium.js 中的图层

在 Cesium.js 中，可以通过添加图层来呈现地球表面的不同类型的数据。以下是一些基本的图层类型和相应的代码示例：

  • 1.ImageryLayer（影像图层）：用于添加卫星影像地图
  • 2.TerrainLayer（地形图层）：用于添加地球表面的高程数据
  • 3.EntityCollection（实体集合）：用于添加自定义的图形元素，如点、线、面等
  • 4.3D Tiles（三维瓦片）：用于呈现大规模的三维模型数据

  • 5.GeoJSONDataSource（GeoJSON 数据源）：用于添加 GeoJSON 格式的数据

除了上述提到的 ImageryLayer、TerrainLayer 和 EntityCollection 之外，Cesium.js 还提供了许多其他类型的图层和功能，如下：
1.BingMapsImageryProvider：用于添加 Bing Maps 的卫星影像地图。

2.ArcGisMapServerImageryProvider：用于添加 ArcGIS Server 提供的地图服务。

3.WebMapTileServiceImageryProvider：用于添加 Web Map Tile Service（WMTS）的地图服务。
