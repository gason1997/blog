# 概述

湖南大屏一张图项目 是一款土地管理类的 GIS可视化软件 ，支持图斑查询，耕地种类获取以及水域查看、路径分析等。
在该项目中本人主要负责耕地保有量模块，主要功能为点击对应行政区时的事件处理、地球与左右两侧可视化图表的联动以及图斑加载与查询等功能（图斑为涉密内部信息，不做展示）

# 主要职责

  • 耕地保有量模块
  • 耕地图斑查询
  • 数据对接

# 功能介绍

  • 本人负责湖南大屏一张图项目的耕地保有量模块功能开发

  • 当鼠标点击右侧 echart 图表的某一项时，地图会自动放大定位对地球对应位置，popup 弹框会显示详细信息

  • 点击空白处时，可移出 geojson 赋值颜色的行政区域，显示行政区边界

  • 点击右侧柱状图时，会显示各区域的耕地面积总量，在地球上以柱子的形式表现，鼠标点击时会高亮显示区域，弹框展示详细信息

  • 点击永久基本农田模块，各区域以耕地总量大小赋值颜色深浅

  • 点击进入详细地州后，边界高亮显示，左侧显示耕地基本信息

  • 可以通过下拉款选取行政区域

  • 点击饼状图加载图斑（图斑为涉密数据不展示）

  • 移除图斑
