# 概述

threejs 展示 BIM 模型的 demo 软件是学习 three 时结合 bim 的练手项目，该项目主要运用 three.js 与 echarts 以及 BIM 模型。实现了大屏的图表与模型展示，支持点击模型查看模型的构件信息。
