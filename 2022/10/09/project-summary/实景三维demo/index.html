# 概述

该实景三维软件 demo 主要功能为加载海量模型，加载 BIM、倾斜投影、三维白膜数据等，支持用户自定义上传和加载模型，自带测量、可视化分析等功能；

# 主要职责

  • 实现用户在后台自定义配置模型数据后加载到 cesium
  • 基于超图平台实现 cesium 基础模块（测量、可视化分析、淹没分析、动态标绘）
  • 基于 vue 实现后台管理系统用户对数据的增删改查
  • 基于 nodeJs+express+mysql 实现后端服务的编写工作
