在 Cesium 中，可以通过自定义 HTML 元素和 Cesium 的弹出框 (Cesium.Popup) 来实现自定义的弹出框（Popup）

# 代码演示

# 步骤解析

  • 首先创建了一个 Viewer 实例。然后，创建了一个自定义的 HTML 元素作为 Popup 的内容，并设置相应的样式。接下来，通过 Cesium.Popup 构造函数创建一个 Popup 实例，并将自定义元素作为内容添加到 Popup 中。
  • 在鼠标移动事件的监听函数中，使用 viewer.camera.pickEllipsoid 方法获取鼠标所在位置对应的地理坐标，然后使用 viewer.scene.cartesianToCanvasCoordinates 方法将地理坐标转换为屏幕坐标。将屏幕坐标设置为 Popup 的位置，使得 Popup 随着鼠标移动而显示在对应位置上。
  • 通过 CSS 样式定义自定义 Popup 的外观，并将样式添加到页面中。将 Popup 的容器元素添加到 Viewer 的容器中，使得 Popup 能够在 Cesium 的场景中显示出来。
