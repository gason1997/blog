# 项目概述

光谷智慧交通系统主要内容分为三部分，第一部分是基于 Mapgis openlayer 框架实现的地图应用，包括地图底图瓦片加载及切换，矢量图层加载与操作，如折线、多边形、矩形、圆形等，地图标注与 popup，地图的动画操作；第二部分是前端数据与后台服务的交互处理，前端通过 vue 的 axios 提交数据到后台，后台与数据库建立连接并操作数据，将处理好的数据返回前端，采用 json 格式做数据交互；第三部分是前端界面搭建以及 UI 设计，前端使用 vue 框架解决操作 html、css 问题，采用 element 框架提升交互体验。

# 主要职责

  • 技术选型（vue+node+mysql+openlayers）前端页面开发等
  • UI 设计 (photoshop+axure)
  • 技术汇报 (ppt)
  • 基于 nodeJs+express+mysql 实现后端服务的编写工作
